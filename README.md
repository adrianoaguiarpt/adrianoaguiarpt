### 👨‍💻 Sobre mim

Tudo bem devs! Sou **Adriano Aguiar**, desenvolvedor full stack em formação, atualmente vivendo no **Porto, Portugal** 🇵🇹

Minha filosofia de estudo foca na **visão Full Cycle:** não apenas escrever código, mas compreender o ciclo de vida completo de uma aplicação. Minha meta é dominar desde a **experiência do usuário (UX)** no Front-end até a **escalabilidade e segurança** das regras de negócio no Back-end, garantindo que cada camada do projeto seja construída com propósito e eficiência.

🎯 **Foco principal:** Back-end, bancos de dados e arquitetura de APIs

🌱 **O que estou estudando atualmente:**
- **JavaScript** (lógica, DOM, eventos)
- **MySQL** (consultas, modelagem de dados)
- Node.js (próximo passo)

🤝 **Aberto a:** Colaborar em projetos open source, trocar experiências e receber feedbacks
